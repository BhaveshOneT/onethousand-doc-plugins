# One Thousand Doc Plugins

Private plugin marketplace for One Thousand GmbH document generation tools, built for Claude Cowork.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **docs-generator** | Scope document generator with One Thousand branding, Graphviz architecture diagrams, confidence scoring, and anti-hallucination verification. EN/DE support. |

## Installation

### 1. Add the marketplace

```
/plugin marketplace add BhaveshOneT/onethousand-doc-plugins
```

### 2. Install the plugin

```
/plugin install docs-generator@onethousand-doc-plugins
```

### 3. Use it

```
/scope-document
```

## Requirements

- Claude Code with plugin support
- GitHub access to `BhaveshOneT/docs-generator-plugin` (private repo)
